# Design Spec: Tycoon Manager Button (Landing Page Placeholder) 🏢✨

**Date**: 2026-03-28  
**Topic**: Добавление кнопки перехода в будущий режим Tycoon Manager.

## 1. Концепция
Добавить интерактивную кнопку «Tycoon» в заголовок (Header) приложения. Кнопка служит визуальной «затравкой» для игроков, обозначая планы на расширение мета-игры (бизнес-менеджмент).

## 2. Дизайн и UI
- **Местоположение**: Справа от текущей кнопки `BonusButton` в `Header.tsx`.
- **Иконка**: `Building2` (из Lucide React).
- **Цветовая схема**:
    - Градиент: `bg-gradient-to-br from-violet-600 to-indigo-700`.
    - Текст/Иконка: Белый (`text-white`).
    - Свечение: Пульсирующая тень фиолетового цвета (`shadow-[0_0_15px_rgba(139,92,246,0.5)]`).
- **Анимация**: Используется `framer-motion` для создания периодического эффекта «дыхания» (пульсация масштаба и свечения).

## 3. Функциональность (Placeholder)
- **Действие**: При нажатии вызывается `toast.info` с сообщением о процессе разработки.
- **Текст сообщения**: *«Разработка Tycoon Manager завершена на 20%. Скоро вы сможете управлять собственной империей! 🔥»*.
- **Состояние**: Кнопка кликабельна, но переход на новую страницу пока не осуществляется (Coming Soon).

## 4. Архитектура изменений
- **Header.tsx**:
    - Добавить компонент `TycoonButton`.
    - Интегрировать `TycoonButton` в десктопную и мобильную версии навигации.
    - Импортировать иконку `Building2`.

## 5. Самопроверка (Self-Review)
1. **Placeholder scan**: Все параметры (иконка, цвета, текст) определены.
2. **Consistency check**: Дизайн дополняет существующий стиль казино, но выделяется цветом, что логично для другого игрового режима.
3. **Ambiguity check**: Местоположение в мобильном меню уточнено.
4. **Scope check**: Задача сфокусирована только на кнопке и визуальной обратной связи.
